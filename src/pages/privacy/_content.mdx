## 2. 取得する情報と利用目的

本サイトでは、訪問者の利用状況を把握しサイト改善に役立てるため、以下の第三者サービスを利用しています。

### 2.1 PostHog（アクセス解析）

[PostHog Inc.](https://posthog.com/)（米国）が提供するプロダクト分析サービスを利用し、ページビュー数、サイト内の遷移、デバイス種別などの集計情報を取得しています。

- 取得する情報: 閲覧したページのURL、リファラ、ブラウザ・OS情報、画面サイズ、IPアドレス（PostHog側で匿名化される設定）、PostHogが発行する匿名識別子
- 利用目的: アクセス傾向の把握、コンテンツ改善
- 取得手段: ブラウザのCookieおよびlocalStorage
- データ送信先: 米国（`us.i.posthog.com`）
- 詳細: [PostHog Privacy Policy](https://posthog.com/privacy)

本サイトは PostHog の `person_profiles: identified_only` 設定を用いており、訪問者をログインユーザーとして識別する処理は行っていません。氏名やメールアドレスといった個人を直接特定する情報は取得していません。

### 2.2 GitHub Pages（ホスティング）

本サイトは [GitHub Pages](https://pages.github.com/)（GitHub, Inc.）でホスティングされており、サーバーアクセスログとしてIPアドレスやUser-Agentが GitHub 側で記録されることがあります。詳細は [GitHub の一般プライバシーステートメント](https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement) を参照してください。運営者は GitHub のサーバーログにアクセスできません。

### 2.3 Adobe Fonts / Google Fonts（Webフォント）

本サイトは [Adobe Fonts (Typekit)](https://fonts.adobe.com/) および [Google Fonts](https://fonts.google.com/) を利用しています。フォント配信のため、訪問者のブラウザから各サービスへリクエストが発生し、IPアドレスやUser-Agentが各社へ送信されることがあります。

- [Adobe Privacy Policy](https://www.adobe.com/privacy/policy.html)
- [Google Privacy Policy](https://policies.google.com/privacy)

## 3. Cookieおよびストレージの利用

本サイトは PostHog の動作のため Cookie および localStorage を使用しています。これらは訪問者の同一セッションを識別するために用いられ、氏名・住所・電話番号など個人を直接特定する情報は格納していません。

ブラウザの設定でCookieを無効にしたり、Cookie/localStorage を削除することで、これらの利用を拒否できます。広告ブロッカー（uBlock Origin など）も多くの場合 PostHog のリクエストをブロックします。

## 4. データの第三者提供

運営者は、訪問者から取得した情報を第三者に販売、貸与、または提供しません。ただし、上記 2 項に記載した各サービス事業者については、それぞれの利用規約・プライバシーポリシーに基づき情報が処理されます。

## 5. データの保管期間

PostHog 上に蓄積されたイベントデータは、PostHog のデフォルト保管期間に従って保存されます。運営者は必要に応じて、過去のデータの削除をリクエストできます。
