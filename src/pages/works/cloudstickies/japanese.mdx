## 特徴
- 付箋はiCloudで同期されます。複数mac間で使用できます。
- 付箋を整理するウィンドウがあり、これはメニューバーから表示することもできます。
- ピン留めした付箋は他のアプリ使用中も常に最前面に表示されます。
- 付箋の色を設定でき、これは端末間で同期されます。

## AI時代の活用例 — プロンプトと出力の一時メモに

Cloud Stickies は、生成 AI（ChatGPT / Claude / Gemini など）を日常的に使うエンジニアやライター、プロダクトマネージャーにとって、以下のような使い方ができます。

### よく使うプロンプトのスニペット置き場として
「コードレビュー用」「議事録要約用」「英訳用」など、繰り返し使うプロンプトのテンプレートを 1 枚 1 プロンプトで付箋にしておけば、ブラウザやチャットアプリの間を行き来するときにコピー＆ペーストの起点として常に手元に置いておけます。色分けで「仕事用」「個人用」「実験中」などを瞬時に区別できます。

### AI 出力の一時的なスクラッチパッドとして
AI が生成したコードスニペットや文章のドラフトを、エディタやドキュメントに貼り付ける前に「いったん寝かせて見比べたい」「複数バージョンを比較したい」というケースは多いものです。Cloud Stickies はそうした **LLM 出力のスクラッチパッド（捨ててもよいメモ）** として機能し、ピン留めしておけば常に最前面に表示されたまま比較作業ができます。

### 複数 Mac での AI ワークフローの同期
自宅の Mac で書きかけたプロンプトの続きを、外出先の MacBook で続ける、といった使い方も iCloud 同期によりシームレスに可能です。AI を使った作業は「思考の断片を素早く保存し、後で別の場所から拾い上げる」ことの繰り返しなので、デバイスを跨いで同期される付箋との相性は非常に良いです。

### Apple 純正「スティッキーズ」との違い
macOS には標準で「スティッキーズ」アプリがありますが、Cloud Stickies は **iCloud 同期** と **メニューバーからのクイックアクセス** を備えており、AI を使った高頻度のメモ取りに最適化されています。

## 使い方
### メニューバーから操作する
  メニューバーからすべての付箋を一覧で確認するウィンドウの表示と、新しい付箋の作成ができます。

![menubar](/stickies-menubar.png)

### 付箋のボタン

| ボタン | 機能 |
|--------|------|
| X | 付箋を閉じる |
| ピンアイコン | 付箋を常に最前面に保つ |
| ^ | 付箋を最小化する |
| ゴミ箱アイコン | 付箋を削除する |
| パレットアイコン | 付箋の色を変更する |
| バインダーアイコン | 付箋の内容をコピーする |

## ダウンロード

[macOS 向け Cloud Stickies をダウンロード（無料 / .dmg）](https://krgpi.github.io/releases/cloudstickies/CloudStickiesInstaller-1.0.4.dmg)

ダウンロードした dmg をダブルクリックでマウントし、`Cloud Stickies.app` を `/Applications` フォルダにドラッグ＆ドロップしてインストールしてください。初回起動時に macOS のセキュリティ警告が表示される場合は、システム設定 → プライバシーとセキュリティから「このまま開く」を選んでください。インストール後は Sparkle により最新版（v1.0.6）に自動アップデートされます。

過去バージョンや zip 版は [Releases ページ](/releases/cloudstickies) からダウンロードできます。
